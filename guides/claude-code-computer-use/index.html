How to Set Up Computer Use in Claude Code

Enable Claude to control your entire desktop, talk to it from your phone, and run tasks on autopilot.

Tools you'll use
Claude Code Desktop

Three new Claude Code features shipped in the same week. Channels let you talk to a running session from your phone. Scheduled tasks run Claude on a recurring schedule without you being there. And computer use gives Claude full control of your desktop: clicking, typing, navigating your apps. Here's how to set up all three.

Channels
1

Talk to Claude Code from your phone

Start a session in Claude Code Desktop. Then open the Claude iOS app or go to claude.ai/code. You can push messages into the running session from anywhere. It's not remote control. It's a direct line into whatever Claude is working on.

Scheduled tasks
2

Set up a recurring task

In Claude Code Desktop, create a scheduled task. Give it a prompt and a frequency (hourly, daily, weekly, or custom cron). Claude runs it on schedule, even if you close the app. Missed runs execute when the app reopens.

Computer use
3

Enable computer use in settings

Open Settings > Desktop app > General and toggle Computer Use on.

4

Grant macOS permissions

  • Accessibility: lets Claude click, type, and scroll
  • Screen Recording: lets Claude see what's on your screen

The Settings page shows the status of each permission. If either is denied, click the badge to open System Settings.

5

Approve apps as Claude requests them

The first time Claude needs to use an app, it asks for permission. You approve per session. Browsers are view-only, terminals are click-only, everything else gets full control.

6

Test it

Ask Claude to do something that requires your desktop. Open an app, interact with a tool that has no CLI, test something in the iOS simulator. Claude controls your screen the way you would.

Before you start

FAQ

Is Claude Code computer use free?
No. Computer use requires a Claude Pro or Max plan. The Claude Desktop app itself is free to download.
What can Claude actually control?
Claude can click, type, scroll, drag, and use keyboard shortcuts in most apps. Browsers are restricted to view-only (Claude uses the Chrome extension for browser work). Terminals and IDEs are click-only. Everything else gets full control.
Does computer use work on Windows?
Not yet. Computer use is currently macOS only. It's a research preview feature.
What are Claude Code channels?
Channels let you push messages into a running Claude Code session from another device. Start a session on your desktop, then send instructions from the Claude iOS app or claude.ai/code.
